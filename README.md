# Rivals ESP

## Features

- **ESP**
  - Boxes (Normal + Corner Boxes)
  - Tracers
  - Player Names
  - Health Bars
  - Distance display
  - Head Dot
  - Rainbow ESP with adjustable speed

- **Team Check**
  - Hides teammates from ESP

- **Customization**
  - Full RGB Color Picker + Hex input
  - Draggable menu with 3 tabs (Colors / Toggles / Config)
  - Import & Export settings

## Keybinds

| Key            | Action                     |
|----------------|----------------------------|
| `Right Shift`  | Open / Close Menu          |
| `Insert`       | Toggle ESP On/Off          |

## How to Use

1. Use a **Roblox Executor**
2. Execute the script
3. Press **Right Shift** to open the menu
